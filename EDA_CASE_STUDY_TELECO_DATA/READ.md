# 📊 Telco Customer Churn Analysis

## 🔍 Project Overview
This project focuses on analyzing customer churn in a telecom company using Exploratory Data Analysis (EDA). The goal is to identify key factors influencing customer churn and provide actionable insights to improve retention.

---

## 📁 Dataset
- 7000+ customer records
- Features include demographics, services, billing, and churn status

---

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy)
- Visualization (Matplotlib, Seaborn)
- Jupyter Notebook

---

## 📊 Key Analysis Performed
- Data Cleaning & Preprocessing
- Univariate Analysis
- Bivariate Analysis
- Multivariate Analysis (Pivot Tables & Heatmaps)
- Correlation Analysis

---

## 🔥 Key Insights
- Customers with month-to-month contracts show highest churn
- New customers (low tenure) are at higher risk
- High monthly charges increase churn probability
- Customers without add-on services are more likely to leave

---

## 💡 Business Recommendations
- Improve onboarding for new customers
- Promote long-term contracts
- Offer bundled services
- Target high-risk customers with retention strategies

---

## 📌 Conclusion
EDA helped uncover critical patterns in customer behavior, enabling data-driven decisions to reduce churn.
