Project Objective

The objective of this project was to analyze consumer complaint data and identify patterns in customer issues, complaint resolution efficiency, and product performance. The project focused on transforming raw complaint data into meaningful business insights using Microsoft Excel.

Dataset Information

The dataset contained around 14,000 consumer complaint records with attributes such as:

Complaint ID
Company Name
Product Category
Issue Type
State
Submission Channel
Date Received
Date Resolved
Timely Response Status
Consumer Dispute Status
Resolution Time
Year and Quarter
Key Analysis Performed
1. Complaint Trend Analysis
Analyzed yearly and quarterly complaint trends.
Identified periods with high complaint volume.
Studied complaint growth patterns over time.
2. Product-Wise Complaint Analysis
Identified products with the highest number of complaints.
Compared complaint distribution across product categories.
Highlighted major customer pain points.
3. Company Performance Analysis
Compared companies based on complaint count.
Evaluated timely response percentages.
Measured complaint resolution efficiency.
4. State-Wise Complaint Analysis
Analyzed complaints by state.
Mapped state codes with state names.
Identified regions with high customer dissatisfaction.
5. Resolution Time Analysis
Calculated complaint resolution duration.
Compared average resolution time across products and companies.
Evaluated operational efficiency.
6. Consumer Dispute Analysis
Analyzed disputed vs non-disputed complaints.
Identified patterns in customer dissatisfaction.

Business Insights Generated
Mortgage and credit-related products generated the highest number of complaints.
Certain companies showed slower complaint resolution performance.
Web channel was the most common complaint submission source.
Some states consistently showed higher complaint volumes.
Timely response rate had a strong impact on dispute occurrence.
