# 🧹 Financial Transactions Data Cleaning Project

## 📌 Overview
This project focuses on cleaning and preprocessing a messy financial transactions dataset using Python (Pandas). The dataset contains real-world issues such as missing values, duplicates, inconsistent formats, and incorrect data types.

---

## 🎯 Objectives
- Handle missing values
- Remove duplicate records
- Standardize categorical data
- Fix data type inconsistencies
- Prepare dataset for analysis

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Jupyter Notebook
